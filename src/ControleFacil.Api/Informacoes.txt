
Entity Framework:
Instalar o dotnet-ef
	Comando:
	dotnet tool install --global dotnet-ef --version 7.0.0-*

Criar as migrations
	Comando:
	dotnet ef migrations add CriarEntidadeUsuario

Executar migrations
	Comando:
	dotnet ef database update

--Passos para criar uma nova entidade (classe):

	1º - Criar o modelo;
	2º - Criar o mapeamento;
	3º - Adicionar mapeamento no ApplicationContext;
	4º - Criar a migration da nova entidade;
	5º - Executar a migration.

--Criando o CRUD:
	1º - Existir o Modelo / Entidade / Classe;
	2º - Criar o repositório;
		- Criar duas pastas, entidades e interfaces;
		- Criar interface(contrato) para toda entidade/repositorio;
	3º - Criar o serviço;
	4º - Criar o controller.
